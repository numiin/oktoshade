# Gruvbox Material Dark, Hard

A port of _Gruvbox Material Dark, Hard_ [theme](https://github.com/sainnhe/gruvbox-material)

Octets:
* _normal_
* _bright_
* _background_
* _shaded_
* _accented_
* _bright accented_

#### NOTE:
Octets 5 and 6 were added to fit the color _Orange_ in from the original scheme.
